# DualShock 3 Flex Contact Spacer Pad

A  replacement spacer pad for Sony DualShock 3 / PS3 controllers.

This part helps support the flexible button membrane/contact film where it presses against the controller PCB. It fixes the most common cause for random button presses, intermittent buttons, or poor membrane contact caused by a worn, flattened, or just plain missing foam spacer.

## Dimensions

Approximate size:

* 22 mm long
* 5 mm wide
* 2 mm thick

## Recommended print settings

* Material: TPU
* Infill: 20%
* Orientation: flat on the bed
* Supports: none

## Notes

This is an aftermarket repair part. It is not an OEM Sony part.

Sony, PlayStation, PS3, and DualShock are referenced only for compatibility identification.
